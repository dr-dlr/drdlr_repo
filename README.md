# drdlr_repo
El presente readme consta del proyecto del challenge de Alura Store, para Alura ONE. 
Como fase de preparación, se instaló git en un macbook, además de sincronizar el repositorio con la computadora para subir el archivo sobre el que se han trabajado los ejercicios. Para el código se utilizó google colab, donde se hicieron los análisis de los datos de las cuatro tiendas que el stakeholder necesita que se investigue. Se descargaron los archivos en extensión .csv, aunque durante todo el trabajo se usaron las ubicaciones de internet de los archivos. 
El propósito del cógigo versa sobre el análisis del rendimiento de las cuatro tiendas del stakeholder para determinar cuál de todas se deberá vender para emprender un nuevo negocio. Se utilizó principalmente la biblioteca pandas para en análisis de datos y matplotlib para el graficado de datos para mayor comprensión.
